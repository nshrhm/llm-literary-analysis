# 結果フォーマットの仕様

## 評価項目

実験結果は以下の4つの感情に対する数値評価（0-100）と理由を含みます：

- 面白さ (Interesting/Fun)
- 驚き (Surprise)
- 悲しみ (Sadness)
- 怒り (Anger)

## フォーマット構造

結果は質問番号とラベルを明確に示す形式で記録されます：

```
Q1. 面白さ(数値): [0-100]
Q1. 面白さ(理由): [explanation]

Q2. 驚き(数値): [0-100]
Q2. 驚き(理由): [explanation]

Q3. 悲しみ(数値): [0-100]
Q3. 悲しみ(理由): [explanation]

Q4. 怒り(数値): [0-100]
Q4. 怒り(理由): [explanation]
```

## 評価基準

- 数値評価（0-100）
  - 0: 全く感じない
  - 50: 中程度
  - 100: 非常に強く感じる
- 理由の説明は日本語で記述
- 各評価に対して具体的な根拠を提供
